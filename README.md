# 🚀 Krazy Studios | Creative Portfolio

A stunning, highly-interactive portfolio website for **Krazy Studios** - a Creative Tech & Design Studio. This app features rich 3D hero animations, modern glassy UI elements, and a dynamic layout designed to showcase services and selected projects.

Made with ❤️ by Alok.

## ✨ Features

- **3D Interactive Hero**: Built with React Three Fiber, featuring a custom interactive "Antigravity" particle system with beautiful brand gradients.
- **Glassmorphism UI**: High-end modern design utilizing glossy surfaces, multi-layered background blurs, and premium aesthetics.
- **Dynamic Animations**: Smooth interactions powered by Framer Motion, custom CSS keyframes, and custom cursors.
- **Responsive Layout**: Fully responsive application optimized for desktop, tablet, and mobile viewing.
- **Premium Typography**: Combining *Schibsted Grotesk* for bold impact and *Lora* for elegant visual accents.

## 🛠️ Tech Stack

- **Frontend Core**: [React 18](https://react.dev/) + [Vite](https://vitejs.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Routing**: [React Router 6 (SPA)](https://reactrouter.com/)
- **Styling**: [Tailwind CSS 3](https://tailwindcss.com/)
- **3D Rendering**: [Three.js](https://threejs.org/) + [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/) + [@react-three/drei](https://github.com/pmndrs/drei)
- **UI Architecture**: [Radix UI primitives](https://www.radix-ui.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 🎨 Modifying & Theming

- **Brand Colors**: Configured centrally via CSS custom properties and the `.gradient-text` utility inside `client/global.css`.
- **Hero Animation Settings**: The 3D particle hero effects can be customized by passing props into the `<Antigravity />` React component found in `client/pages/Index.tsx` (e.g. `particleSize`, `colors`, `autoAnimate`, `fieldStrength`).
